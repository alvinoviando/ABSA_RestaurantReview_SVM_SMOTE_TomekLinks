Training Data

Silahkan lakukan proses training dengan memasukkan data training agar model dapat belajar mengidentifikasi sentiment suatu rumah makan atau restoran.
Instruksi :
  • Admin harus menyiapkan data training dengan ekstensi .csv dan mengikuti format file csv dengan kolom sesuai dengan file berikut ini
  • Setelah menyiapkan data, admin meng-upload data tersebut dengan memilih "Choose File" dan pilih "Train Data"
  • Proses training akan berlangsung. Setelah proses training selesai, hasil training akan ditampilkan
  • Jika admin ingin mencari sentimen terkait suatu rumah makan atau restoran , admin dapat memilih menu "User Input"
